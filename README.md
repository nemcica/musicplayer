# Music Player

## Description

A simple desktop application developed in **C++** using **Qt Creator** and **SFML** library, that currently supports only .mp3 files.

## Features

- **Load Songs**: Import songs from a specified folder.
- **Playback Controls**: 
  - Play
  - Pause
  - Stop
  - Rewind
  - Next Song
- **Playlist Management**: 
  - Shuffle
  - Loop
  - Queue Songs
